SavApp - Calculateur de saponification

Développement d’une application web permettant de calculer la composition de savons en saponification à froid (SaF). Le projet repose sur des algorithmes métier intégrant les indices de saponification, le surgraissage et les propriétés des corps gras.

L’application fonctionne avec une API REST externe permettant de gérer les ingrédients, utilisateurs et données métier. Cette API, développée en Kotlin avec Spring Boot, est essentielle au bon fonctionnement du calculateur.

Kotlin Spring Boot Angular MariaDB API REST Calcul métier
